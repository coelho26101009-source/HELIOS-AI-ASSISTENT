━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  O HELIOS pede SEMPRE confirmação antes de:

  🔴 PERIGO ALTO (sempre pede):
    • Apagar ficheiros permanentemente
    • Formatar drives
    • Escrever no registo do Windows
    • Matar processos do sistema

  🟡 PERIGO MÉDIO (pede se o conteúdo for sensível):
    • Comandos PowerShell com: Remove-Item, Format, del, rd,
      rmdir, Stop-Process, shutdown, Restart-Computer, Clear-Disk
    • Submeter formulários web (comprar, pagar, confirmar, apagar)
    • Mover ficheiros em massa

  Como responder à confirmação:
    → Aparece um modal na interface com CONFIRMAR / CANCELAR
    → Tens 60 segundos para responder
    → Se não responderes, a ação é cancelada automaticamente

  ⚙️  CONFIGURAÇÃO  (config/settings.yaml)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Ficheiro .env (variáveis privadas):
    GROQ_API_KEY=...          → Chave API do Groq (obrigatória)
    ELEVENLABS_API_KEY=...    → Chave ElevenLabs (opcional, voz humana)
    OPENAI_API_KEY=...        → Chave OpenAI (opcional)

  settings.yaml (comportamento):
    groq_model: "llama-3.1-8b-instant"   → Modelo de IA (rápido)
    tts_enabled: true/false               → Liga/desliga voz
    tts_provider: edge/elevenlabs/openai  → Motor de voz
    edge_voice: "pt-PT-DuarteNeural"      → Voz portuguesa masculina
    edge_voice: "pt-PT-RaquelNeural"      → Voz portuguesa feminina

  🔌 COMO TRANSPORTAR O HELIOS (pen/portátil)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Para levar o HELIOS numa pen ou noutro PC:

  1. Copia a pasta HELIOS_V7 para a pen
  2. No PC de destino, garante que tens:
     → Python 3.11+ instalado (python.org)
     → Google Chrome instalado
  3. Abre o terminal na pasta HELIOS_V7
  4. Instala dependências: pip install -r requirements.txt
  5. Corre: .\start.bat

  Para criar um .exe portátil (sem precisar de instalar nada):
  → Corre no terminal: pip install pyinstaller
  → Corre: pyinstaller --onefile --noconsole core\main.py

  🐞 RESOLUÇÃO DE PROBLEMAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  "Não responde / demora muito"
    → Verifica ligação à internet (Groq precisa de internet)
    → Verifica se a API key está correta no .env
    → Vê o log: helios.log

  "Rate limit atingido"
    → Aguarda 30-60 segundos e tenta de novo
    → O modelo llama-3.1-8b-instant tem limites maiores

  "Sem voz"
    → Verifica se tts_enabled: true no settings.yaml
    → Instala: pip install edge-tts pygame

  "Erro no plugin X"
    → Vê detalhes em: helios.log
    → Playwright (web_vision): playwright install chromium

  Log completo: HELIOS_V7\helios.log
